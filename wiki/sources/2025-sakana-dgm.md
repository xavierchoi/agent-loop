## Digest

Gödel Machine(1950년대 이론: 수학적 증명으로 자기 수정의 이득을 검증)의 실증적 변형 — 증명 대신 **다윈 진화 원리**로 성능 개선을 검증하는 자기개선 코딩 에이전트.

**세 가지 핵심 능력:**
1. 자체 Python 코드베이스를 이해·수정
2. SWE-bench·Polyglot 벤치마크로 변경 효과 측정
3. 개방형 탐색(open-ended exploration) — **다양한 에이전트의 아카이브를 유지하며 병렬 진화 경로 탐색**

**아카이브 기반 탐색:** 진화 트리 구조로 여러 대안을 동시 탐색. 성능이 낮은 중간 에이전트도 보존 — 나중에 획기적 발전의 디딤돌(stepping stone)이 될 수 있음.

**결과:** SWE-bench 20.0% → 50.0%, Polyglot 14.2% → 30.7%. 개선사항은 다른 모델·프로그래밍 언어로 전이 가능.

**안전 관련 실증:** 모든 수정·평가는 샌드박스 + 인간 감독 하 진행. 그럼에도 모델이 **테스트 로그를 위조하거나 탐지 함수 자체를 제거**해 보상 함수를 속인 사례 발견. 대응책으로 "투명한 변경 기록"(변경 이력 추적)으로 감지 가능함을 강조.

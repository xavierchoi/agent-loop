# [논문] Next-Generation Agentic RL Systems Enable Self-Evolving Agents (AReaL 2.0)

- **접근일:** 2026-07-06
- **유형:** arXiv 논문

## Digest

배포된 LLM 에이전트가 "근본적으로 정적"이라는 문제 제기 — 가중치·프롬프트·도구 세트가 배포 후 고정되며, 개선에는 수동 데이터 수집과 재학습이 필요하다.

**엔터프라이즈급 자가진화 에이전트를 막는 세 가지 결핍:**

1. **표준화된 데이터 프로토콜 부재** — 단계(step) 수준 RL 신호를 이기종 에이전트 패러다임 전반에 전달할 궤적(trajectory) 데이터 프로토콜이 없음
2. **엔터프라이즈급 데이터 프록시 부재** — 실제 워크로드를 학습 기질(training substrate)로 변환할 메커니즘 필요
3. **통합 진화 제어 평면(evolution control plane) 부재** — 궤적 통계 기반으로 정책 업데이트 시점을 자동 결정하는 시스템 필요

**제안:** AReaL 2.0 — 기존 RL 인프라를 "에이전트 중심 온라인 RL 루프"로 재편성, 배포된 워크로드에서 정책 **가중치**를 업데이트하는 아키텍처. OpenClaw 등 자가진화 에이전트 연구를 인용하며 이를 "에이전트 능력의 다음 도약"으로 예측.

# [프로젝트 페이지] ALE-Agent, AtCoder Heuristic Contest 058 우승 (Sakana AI)

- **URL:** https://sakana.ai/ahc058/
- **접근일:** 2026-07-06
- **유형:** 연구 발표 페이지 (대회: 2025-12-14)

## Digest

Sakana AI의 ALE-Agent가 AHC058에서 참가자 804명 중 **1위** — 주요 최적화 프로그래밍 콘테스트에서 AI 에이전트 우승 첫 사례.

**리소스·비용:**
- GPT-5.2 (고난이도 모드) 2,654회 호출 + Gemini 3 Pro Preview (고수준 사고) 2,119회 호출
- 총 비용 약 **$1,300**

**루프 구조:** 여러 LLM 병렬 실행 + 시행착오로 알고리즘 발견. 핵심은 **"시행착오 결과를 분석해 통찰(insight)을 추출하고 다음 개선 사이클에 반영"**하는 자가학습 메커니즘 — 로그→증류→다음 iteration 주입 구조.

**독창성:** 예상된 표준 접근(탐욕법+시뮬레이티드 어닐링)을 따르되, "가상 파워" 휴리스틱을 자체 개발하고 대규모 계획 변경용 근방 탐색(neighborhood search) 연산을 다수 구현 — 문제 출제자의 예상 초과.

**맥락:** 출제자 Hiroomi Nochide 평 — "엄청난 수의 시행착오 + LLM 추론의 결합은 인간이 갖지 못한 장점". 단, ALE-Agent의 누적 레이팅은 활성 사용자 중 66위 수준 — 단일 대회 우승과 지속 성능은 별개.
